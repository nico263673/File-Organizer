File Organizer

A simple Python project that scans a folder and categorizes files based on their file extensions.

What this project does

This program:

Reads all files in a selected folder
Detects file types using file extensions
Sorts files into categories
Prints a structured overview of the files
Supported file types
Images
.jpg, .jpeg, .png, .gif, .webp
PDFs
.pdf
Music
.mp3, .wav, .flac
Videos
.mp4, .mkv, .mov
Text files
.txt, .md
Archives
.zip, .rar, .7z, .tar, .gz
Current version (v1)
